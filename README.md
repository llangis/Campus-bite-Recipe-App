# Campus Bites Recipe Web Application

<img src="src/mvc2025TermProject/wwwroot/img/Campus Bites Logo.png" alt="Campus Bites Logo" width="200">

A student recipe web app built with **ASP.NET Core MVC** for easy, affordable meal ideas.

## 📖 Project Overview

**Campus Bites** helps students discover, create, and share affordable, easy-to-cook meals.  
The app supports recipe creation, ingredient and category management, and simple authentication for registered users.

**Catch Phrase:** "Easy Meals for Student Life"
